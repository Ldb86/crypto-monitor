🔧 KEY FEATURES OF NEW STRUCTURE
─────────────────────────────────

1. **Independent Services**
   - Each bot runs in separate process
   - Can scale individually
   - Separate logs
   - Independent crash recovery

2. **Railway Compatible**
   - Each service = 1 Railway service
   - Own environment variables
   - Own start command
   - Auto-deploy on git push

3. **Shared Resources**
   - Reference code in shared/
   - Utilities separated
   - Easy to reuse across services

4. **Clean Repository**
   - .gitignore configured
   - .env excluded (security)
   - No node_modules tracked
   - Backup files archived

5. **Documentation**
   - Main README with overview
   - Service-specific READMEs
   - Environment setup guide
   - Railway deployment guide
